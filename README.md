Financial Health Dashboard

Overview

This project is an interactive Financial Health Dashboard built using Power BI. It provides insights into an organization's financial performance by analyzing revenue, expenses, profitability, assets, liabilities, and cash flow.

Objectives

- Monitor key financial metrics.
- Analyze revenue and profit trends.
- Evaluate financial stability through balance sheet analysis.
- Track cash inflows and outflows.
- Generate business insights for decision-making.

Tools & Technologies

- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)

Dashboard Features

- KPI Cards for Revenue, Expenses, Net Profit, and Profit Margin
- Revenue Trend Analysis
- Profitability Trend Analysis
- Balance Sheet Overview
- Cash Flow Analysis
- Interactive Dashboard Visualizations

Key Insights

- Revenue shows a steady growth trend.
- Net profit remains positive across the analysis period.
- Assets consistently exceed liabilities.
- Cash inflows generally surpass cash outflows.
- The organization demonstrates strong financial health.
